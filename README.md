# Ai-Social-Media-Post-Creator
AI Social Media Post Creator is a simple web application that generates engaging captions and hashtags for social media posts using AI. Users can enter a topic, select the platform and tone, and instantly receive multiple caption ideas. Built using Python and Streamlit with Gemini AI integration.
This project is an AI-powered Social Media Post Creator that helps users generate captions and hashtags for different platforms. The application allows users to input a topic, choose the platform and tone, and receive AI-generated content instantly.

Technologies Used:
- Python
- Streamlit
- Gemini API
